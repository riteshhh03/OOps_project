This is a simple Python project that demonstrates Object-Oriented Programming (OOP) concepts using a real-world example.

The system allows a customer to rent vehicles such as Car and Bike and calculates the total rental cost based on the number of days.

OOP Concepts Used :- 

1. Classes & Objects to model vehicles and customers

2. Inheritance to create different vehicle types

3. Polymorphism for different rent calculations

4. Encapsulation to protect vehicle data
